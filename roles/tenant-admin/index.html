Tenant Admin

As a Tenant Admin you own the organisation. You set MedOps up, decide
how it works for everyone else, and keep an eye on the whole operation.

Your reach

You can manage everything across your organisation and all of its
hospitals — the structure, the people, the catalogue and the preferences.
Day-to-day clinical and front-desk work is usually done by doctors and
operations staff, but you can always see what’s happening.

Setting up the organisation

Your first job is to build out the structure. A typical order is:

  1. Create your hospitals
    Add each hospital with its name, address, licences and departments.
    Hospitals & Departments →
  2. Add departments
    Create the clinical departments inside each hospital.
  3. Build your wards
    Add floors, rooms and beds, and set room pricing (tariffs), so patients
    can be admitted later.
  4. Add your people
    Invite doctors and operations (front-desk) staff by email.
    Managing Doctors → ·
    Operations Team →
  5. Set up the pharmacy
    Create one or more pharmacies and add pharmacy logins.
    Pharmacies →
  6. Tune your preferences
    Set branding, document code prefixes, tax details and inpatient billing
    rules.

Organisation preferences

You control settings that affect everyone:

  • Branding & identity — each hospital’s display name, address and logo.
  • Document code prefix — a short prefix (for example, RK) that seeds
    the reference codes on prescriptions, invoices and other documents.
    Changing it only affects new documents.
  • Tax details — GST identification for hospitals, and tax rates for
    each pharmacy.
  • Inpatient billing rules — how bed-day charges are counted (by
    calendar day or rolling 24 hours) and the grace period before another
    day is charged.
  • Inactivity sign-out time — how long before idle users are signed
    out, from 1 minute up to 24 hours (the default is 15 minutes).

Keeping oversight

  • Activity log — see who did what and when across the organisation,
    with search and date filters. Open →
  • Global search — quickly find patients, doctors, hospitals,
    departments, appointments, bills and medicines.
